A simple and light-weight way to back/scroll to top.

== Description ==
Scroll Top plugin by inthiscode allows the visitors to smoothly scroll back to the top of the page with "**easing**" animations, with fully customizable options and image. WPFront Scroll Top plugin has the following features.



Please visit [Scroll To Up by InThisCode's Demo](https://www.inthiscode.com/scroll-to-up/).

### Features of WordPress Scroll To Top plugin

*   **Fully Responsive**
*   **Fully customizable options pages**
*   **Works on all major browsers**
*   **Tested with all versions of WordPress**
*   **All Easing Types. For more look at [easings.net](http://easings.net/)**
*   **Animations (fade, slide)**
*   **3 Styles (image, link, and pill)**
*   **30+ different icons to choose from**
*   **Custom URL Option. Set your own custom image icon**
*   **Location Settings: bottom right, bottom left, top right, top left**
*   **Option to display "back to top" only on homepage**
*   **Option to display "back to top" only on posts (post type=>post)**
*   **Option to display only on pages**
*   **Option to display only on category archives**

== Installation ==
Install using Plugins->Add New or manually upload using WordPress plugin uploader.
Activate the plugin, and go to the settings page for customization.
